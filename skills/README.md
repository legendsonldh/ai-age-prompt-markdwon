# Skills

This directory contains reusable Skill packages.

## Available Skills

- [tauri-apple-glass-ui](tauri-apple-glass-ui/README.md): Apple-inspired light glass UI language for Tauri desktop clients.
- [tauri-client-build-packaging](tauri-client-build-packaging/README.md): macOS/Windows Tauri build, sidecar, signing, and packaging automation.
- [go-react-admin-platform](go-react-admin-platform/README.md): Go/Gin/GORM backend with React/Vite admin frontend patterns.
- [workspace-package-management](workspace-package-management/README.md): Go/Rust/npm workspace and dependency management patterns.
- [wpf-aurora-ui](wpf-aurora-ui/SKILL.md): .NET 10 WPF desktop UI design system with token-based Aurora styles, DataGrid patterns, MVVM with CommunityToolkit.Mvvm, and production GUI standards. Ideal for multi-project WPF solutions needing consistent appearance.

Each folder is self-contained and includes its own `SKILL.md`.
